# Server Side

## Technologies
```
1. Node.js
2. GraphQL
3. Apollo-server
```

## Getting Started

1. Clone the project:
`git clone https://github.com/AnguleMathias/TshirtShop.git`

2. Change Directory to server:
`cd TshirtShop/server `

3. Add dependencies:
`npm install`

4. Run server: 
`yarn start`

Enjoy :rocket:
